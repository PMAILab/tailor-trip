# Task: Implement Mood Cards Component

**Feature Area:** Mood-Based Discovery
**Status:** Todo
**Priority:** High

## Description
Develop a scrollable UI component for "Mood Cards". These cards represent the different feelings a user might want from their trip.

## Predefined Options
- Need a reset
- Adventure mode
- Budget weekend
- Romantic escape
- Workation vibe
- Explore something new

## Acceptance Criteria
- [ ] Create a reusable `MoodCard` component.
- [ ] Implement a horizontal scrollable container for the cards.
- [ ] Add distinct visuals, icons, or background colors for each predefined mood.
- [ ] Ensure accessibility (ARIA labels) and smooth scrolling behavior.
