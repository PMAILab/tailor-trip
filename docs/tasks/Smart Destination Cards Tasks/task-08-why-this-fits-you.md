# Task: Add "Why This Fits You" Expandable Section

**Feature Area:** Smart Destination Cards
**Status:** Todo
**Priority:** High

## Description
Implement an expandable "Why this fits you" section within each `DestinationCard`. This provides a personalized explanation of why the destination was recommended, generated by the Gemini API based on the user's mood, budget, season, and other contextual factors.

## Example
> "Recommended because it's affordable in July, low crowd season, and matches your weekend duration."

## Acceptance Criteria
- [ ] Add a collapsible section at the bottom of each `DestinationCard`.
- [ ] Default state is collapsed, showing a teaser line or a "Why this fits you" toggle.
- [ ] Expanded state reveals the full AI-generated explanation text.
- [ ] Animate the expand/collapse transition smoothly (height + opacity).
- [ ] Ensure the `whyItFitsYou` field is populated from the Gemini API response.
- [ ] Handle empty or missing explanation text gracefully (hide the section).
